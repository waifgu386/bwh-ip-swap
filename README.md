**操作步骤：**

2. 左侧菜单找到 **Migrate to another DC**

## 换IP之后要做什么

拿到新IP不是故事的终点，还有几件事要处理：

**1. 更新SSH连接配置**

如果你本地保存了旧IP的 SSH 连接，记得改成新的。

**2. 更新域名DNS解析**

网站解析到旧IP的话，改 DNS 记录指向新IP。DNS 传播通常需要几分钟到几小时，不要急。

**3. 更新VPS上绑定IP的应用配置**

如果你的程序里写死了 IP 地址，逐一检查更新。

**4. 新IP连不上怎么办**

拿到新IP后用 ping 或 SSH 测试。确实还是连不上，先确认等待时间够了（有时需要几分钟），再考虑联系客服——你已经付了钱，客服会确保给你一个可用的IP。

## 关于搬瓦工换IP的常见问题

**Q：换IP会不会丢数据？**

不会。换IP只是更换服务器的网络地址，VPS上的系统、文件、数据库全部保留不变。

**Q：迁机房和付费换IP有什么区别？**

迁机房是把你的VPS搬到另一个地理位置的数据中心，IP自然变了，费用为零，数据保留；但如果IP已经被封，这个入口会被锁死。付费换IP是在当前机房分配一个新IP，不移动VPS，$8.79一次，适合IP被封后的补救。

**Q：哪个套餐最适合担心IP被封的用户？**

**Q：搬瓦工换IP有次数限制吗？**

付费换IP没有次数限制，钱到账就处理。迁机房也没有次数限制，但每次迁移会消耗一定月流量（迁移本身产生的数据算进去）。
